# Temperature Converter (Java)

## Overview

This is a simple Java console application that converts temperatures between Celsius and Fahrenheit. It allows the user to input a value in one unit and see the equivalent in the other.

## Features

* Converts Celsius to Fahrenheit
* Converts Fahrenheit to Celsius
* User-friendly console input/output

## How It Works

1. The program prompts the user to enter a temperature in Celsius.
2. It converts the value to Fahrenheit and displays the result.
3. Then, it asks for a temperature in Fahrenheit.
4. It converts that value to Celsius and displays the result.

## Formula Used

* **Celsius to Fahrenheit:** `(C × 9/5) + 32`
* **Fahrenheit to Celsius:** `(F − 32) × 5/9`

## Requirements

* Java (JDK 8 or higher)

## How to Run

1. Compile the program:

   ```
   javac convertTemp.java
   ```
2. Run the program:

   ```
   java convertTemp
   ```

## Example

```
Enter temperature in Celsius: 25
25.0 degrees Celsius is equal to 77.0 degrees Fahrenheit.

Enter temperature in Fahrenheit: 77
77.0 degrees Fahrenheit is equal to 25.0 degrees Celsius.
```
